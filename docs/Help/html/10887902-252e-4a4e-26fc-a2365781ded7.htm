GameManager PropertiesProject Zodiac Documentation
The GameManager type exposes the following members.
Properties
  NameDescription
Awesomeness
Gets or sets the overall awesomeness
Checkpoints
Gets or sets Checkpoints in the level
CurrentLevel
Gets or sets the current level.
CurrentSection
Gets or sets the current section of the current level.
Difficulty
Gets or sets the game difficulty. Higher integer means harder difficulty.
GameMode
Gets or sets the gamemode.
HeroCount
Gets or sets the Number of heroes that should spawn in the entire level.
Items
Gets or sets items in the level.
LevelNo
Gets or sets the number of the level to generate/load.
LevelNoMultiplayer
Gets or sets the number of the level to generate/load.
LevelTime
Gets the time that has passed since the level has started. (Statistics) Set when the level ends.
LosingTeam
Gets the losing team that caused the game to end.
Mercenaries
Gets or sets mercenaries, even those not associated with a team.
MercenaryCount
Gets or sets the Number of mercenaries that should spawn in the entire level.
MobCount
Gets or sets the Number of mobs that should spawn in the entire level.
NetworkController
Gets the NetworkController whch holds the network connection logic
NextMercenaryId
Gets or sets the next available mercenary ID.
NextMobId
Gets or sets the next available mob ID.
PlayerCount
Gets or sets the Number of player for the mode.
PlayFirstTrigger
Gets or sets a value indicating whether section 1 clearance is triggered in Play-Mode
PlaySecondTrigger
Gets or sets a value indicating whether section 2 clearance is triggered in Play-Mode
PlayThirdTrigger
Gets or sets a value indicating whether section 3 clearance is triggered in Play-Mode
SceneToBeLoaded
Gets or sets the scene that will be loaded.
SectionNo
Gets or sets the number of the section to start.
TeamCount
Gets or sets the Number of teams, neccessary for dynamic modes.
